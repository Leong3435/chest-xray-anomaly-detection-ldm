# Chest X-ray Anomaly Detection using Latent Diffusion Model

This project explores unsupervised anomaly detection for chest X-ray images using a Latent Diffusion Model.

## Overview
The model is trained primarily on normal chest X-ray images and detects abnormalities by measuring reconstruction differences between the input image and generated image.

## Methods
- Variational Autoencoder (VAE)
- Latent Diffusion Model
- Reconstruction-based anomaly scoring
